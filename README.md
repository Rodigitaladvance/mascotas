# AURA Pets | Ecosistema de Excelencia Sanitaria Global

![AURA Banner](/brand/aura-logo.png)

AURA Pets es la plataforma definitiva de gestión biométrica y sanitaria para mascotas de alta distinción. Diseñada bajo la filosofía de **"Quiet Luxury"**, combina una estética de grado banca privada con protocolos de seguridad médica internacional.

## 💎 Pilares del Ecosistema

### 1. Protocolo de Localización Inteligente
Sistema adaptativo que configura automáticamente el entorno (Idioma, Divisa y Unidades) mediante la detección de zona horaria, permitiendo además el ajuste manual para viajeros frecuentes y expatriados premium.
- **Regiones Optimizadas**: España (€/kg), Australia ($/kg), USA ($/lbs), UK (£/kg).

### 2. Bóveda de Privacidad AURA™ (Bio-Security)
Arquitectura de seguridad de grado médico que garantiza la soberanía del dato del cliente.
- **Cifrado AES-256**: Los registros están protegidos localmente con encriptación simétrica.
- **Cumplimiento Global**: Estructura alineada con **GDPR (Europa)** y **HIPAA (USA)**.
- **Derecho al Olvido**: Sistema de destrucción permanente de datos integrado.

### 3. Inteligencia Multi-Especie
Módulos dinámicos adaptados a la biología específica de cada miembro:
- **Equinos**: Historial de herraje, rendimiento deportivo y registro REGA.
- **Exóticos**: Monitorización de hábitat mediante **Cronógrafos Analógicos** de precisión para Temperatura y Humedad, y control de ciclos de muda.
- **Aves**: Registro de anillado, ciclos de canto y muda de pluma.

### 4. Modo SOS & Global Passport
Interfaz de alta visibilidad para emergencias críticas y sistema de logística transfronteriza para movilidad internacional garantizada.

## 🛠️ Especificaciones Técnicas

- **Frontend**: React + Vite (Arquitectura de Alto Rendimiento).
- **Estética**: Glassmorphism Elite + Tipografía Serif de Lujo.
- **Iconografía**: Lineal Minimalista (Lucide React).
- **Animaciones**: Movimiento Cinematográfico (Framer Motion).

## 🚀 Despliegue

El sistema está optimizado para funcionar tanto en entornos locales como en despliegues globales. Todos los activos visuales (Logo) se sirven mediante rutas relativas (`/brand/`) protegidas por el protocolo de carga de AURA.
